# 多态

多态，就是说多种形态。

在 CS 中，玩家可以购买的武器有很多种，但它们都是枪，都能开火和换弹，只是开火方式和换弹动作不同，这时我们就可以定义一个枪类，在其中实现开火和换弹方法，接下来为每种型号的枪创建一个枪类的子类，并重写开火和换弹方法以实现不同的武器效果。

多态在这里的作用就是，不论玩家手持哪把武器，外部代码都可以按照枪类的调用方式调用不同武器的开火和换弹方法。代码中的体现是：**子类实例可以存储到父类类型的变量中，并且可以访问父类中存在的成员**。

这里来看看代码例子：

```gdscript
# 这里是 枪 类
class_name 枪

func 开火():
    pass

func 换弹():
    pass
```

```gdscript
# 这里是 AK47 类
extend 枪
class_name AK47

func 开火():
    print("AK开火")
```

```gdscript
# 这里是 AWP 类
extend 枪
class_name AWP

func 开火():
    print("AWP开火")
```

此时，当我们获取到一把 AWP 或 AK47 时，我们都可以把它们存放到 枪 类型的变量中，并对这把枪调用开火或换弹。

多态的作用就是，我们只需要知道手上是一把枪就能开火和换弹，而不用关心枪的型号，至于具体开火和换弹的方式，则是由枪的子类自己决定。
