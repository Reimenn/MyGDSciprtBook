认识面向对象

面向对象（简称OOP），最流行的编程方式。

对于游戏开发而言，面向对象编程方式是最符合人类逻辑的，甚至给你一种编程也是玩游戏的感觉。

这一章内容不多，但都是理论概念上的东西，没有啥整活章节，可能略枯燥，不过下一章整活丰富，希望各位坚持看完本章。
